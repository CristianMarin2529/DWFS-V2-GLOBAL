# Repaso fundamental de HTML y CSS

HTML y CSS son los lenguajes fundamentales para la creación de páginas web. HTML (HyperText Markup Language) estructura el contenido, mientras que CSS (Cascading Style Sheets) se encarga del diseño y la presentación visual.
