# Repaso fundamental de API REST

Las API REST (Representational State Transfer) son un estándar para la creación de servicios web que permiten la comunicación entre aplicaciones de forma sencilla, escalable y basada en recursos. Son ampliamente utilizadas en el desarrollo backend y frontend para integrar sistemas y aplicaciones. RestFul APIs utilizan HTTP como protocolo de comunicación y se centran en recursos identificados por URLs, utilizando métodos HTTP (GET, POST, PUT, DELETE) para interactuar con esos recursos, así como sustantivos para identificar los recursos.
