# Repaso fundamental de DOM

El DOM (Document Object Model) es una interfaz de programación que representa la estructura de los documentos HTML y XML. Permite manipular dinámicamente el contenido, la estructura y el estilo de una página web mediante lenguajes como JavaScript.
